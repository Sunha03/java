< 생성자 >
생성자 => 메소드.
객체가 생성될 때 호출되는 특별한 메소드이며 주로 자원을 액세스하거나,
자원을 사용하기 위해서 초기화하거나, 자원 관련 작업을 하거나, 객체가 가지고 있는 멤버 변수를 초기화하는 작업을 정의
자원 : DBMS, 네트워크, 파일 시스템, ...

[객체 생성]
클래스 타입 변수 = new 생성자 메소드()
> 사용할 클래스              > 클래스 안에 미리 정의되어 있는 생성자 메소드를 호출.
						일반 메소드처럼 생성자 메소드도 매개변수로 외부에서 값을 전달 받아 사용할 수 있음
[규칙]
1. 생성자 메소드는 클래스명과 대소문자까지 정확하게 동일한 이름으로 정의해야 함
   > 리턴타입 명시 (X)
2. 생성자 메소드를 정의하지 않으면 컴파일러가 기본 생성자를 제공함
   > 기본 생성자 : 매개변수가 없는 생성자
   > 생성자 메소드를 개발자가 정의하면 컴파일러가 기본 생성자를 제공하지 않음
   > 처리되는 일이 없다고 하더라도 무조건 기본 생성자는 정의해야 함
3. 생성자 메소드도 일반 메소드처럼 매개변수를 정의하고 외부에서 값을 전달받아 사용할 수 있음
   > 주로 객체에 정의된 멤버변수의 값을 초기화하는 작업
4. 생성자 메소드도 일반 메소드처럼 오버로딩 적용 (O)
5. 생성자 메소드 내부에서 다른 생성자 메소드를 호출할 수 있다.
   > this(매개변수, ...)
   > 반드시 첫 번째 문장에서 호출해야 함
